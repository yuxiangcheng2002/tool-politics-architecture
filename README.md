# Tool Politics in Architecture
## AI Image Generation as Ideological Infrastructure

Deep research notes expanding Yuxiang Cheng's 'Interfacing Language Model' (2025) into architectural AI tool critique. This repository serves as a textbook-style companion for a critical theory assignment in architecture.

---

## Overview

AI image generation systems in architecture (Midjourney, Stable Diffusion, DALL-E) are not neutral productivity tools. They are ideological machines—encoding aesthetic defaults, political economies of labor, environmental costs, and epistemological conditions—that restructure architectural imagination, authorship, and representation from within. 

By applying media theory, psychoanalysis, and political economy, this research maps how the "prompt" inherits both the performative force of a speech act and the machinic authority of executable code, while the latent space colonizes the architect's visual imagination with statistically averaged aesthetics.

## Core Framework (Yuxiang Cheng, 2025)

This research builds upon the framework established in **"INTERFACING LANGUAGE MODEL: From Production with Use to Productive Use"** (NYU MCC, Prof. Galloway, May 2025), translating the critique of LLMs into the spatial/visual domain of architecture:

| LLM Concept | Architectural Translation |
|-------------|---------------------------|
| Textual general intellect | Visual/Spatial general intellect (LAION-5B) |
| Staff Experience Designer | AI Visualization Specialist / Prompt Engineer |
| Text prompt to text output | Text prompt as corrupted design brief to image output |
| Linguistic hallucination | The Promptable Uncanny (structural/spatial impossibility) |
| The Blackboxed LLM | The Proprietary Latent Diffusion Model |
