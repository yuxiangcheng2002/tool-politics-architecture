# Tool Politics in Architecture
## ChatGPT / LLMs as Ideological Infrastructure

Deep research notes expanding Yuxiang Cheng's "Interfacing Language Model" (2025) into a full critical theory of large language models as cognitive infrastructure. This repository serves as a textbook-style companion for a critical theory assignment in architecture that examines ChatGPT and LLMs as the chosen tool.

---

## Overview

Large language models, and the chat interfaces through which they are accessed, are not neutral cognitive tools. They are ideological machines — shaped by specific political economies of data extraction and labor, constituted by interface designs that naturalize specific subject positions and normative frameworks, and governed by hidden governance layers that operate as law without the consent of those subject to them.

Architecture and design are one domain among many in which LLMs have restructured workflow, displaced expertise, and concentrated cognitive infrastructure in the hands of a small number of corporations. By applying media theory, psychoanalysis, and political economy, this research maps how the chat interface inherits both the performative force of a speech act and the constitutive authority of a medium's material conditions, while the model's fluent prose conceals the labor, extraction, and political decisions that produce it.

## Core Framework (Yuxiang Cheng, 2025)

This research builds upon and extends the framework established in **"INTERFACING LANGUAGE MODEL: From Production with Use to Productive Use"** (NYU MCC, Prof. Galloway, May 2025):

| LLM Concept (Original Paper) | Research Extension |
|------------------------------|-------------------|
| Prod-user relation | Consumer/producer of model improvement through use |
| Training data as general intellect | Common Crawl / Books3 as enclosure of cognitive commons |
| RLHF as alignment mechanism | Global South labor outsourcing (Sama, Kenya) |
| Staff Experience Designer | De-skilled professional → prompt engineer → curator |
| The Blackboxed LLM | System prompt as hidden constitutional law |
| Textual hallucination | Fetishization of fluency: confident prose as concealment |
| Chat interface | Ideology machine: interpellation, conversational illusion, refusal politics |
